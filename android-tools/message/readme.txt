备注：tool需要运行在windows平台

1.pduspy tool
  pduspy针对GSM 3gpp sms

  a、运行PDUspy.exe
  将pdu copy到manual Enter message编辑框，
  根据接收短信还是发送短信点选sms type

  b、点击“decode” 按钮
